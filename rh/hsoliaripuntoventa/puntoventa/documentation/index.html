  • Installation

    step 1

    Upload all files and folders which are inside the "ZAP POS" folder from the .zip file you have downloaded from Codecanyon to your web using cPanel or FTP program (ie. FileZilla).

    step 2

    Make sure that The folder "files" have write permissions (777) (follow the image).

    step 3

    Open the install folder http://yourdomain.com/pathtoupload/install/ in your web browser

    step 4

    Pre-intall system checklist - If there is any issue with checklist, you will need to fix it first to proceed to next step.

    step 5

    Provide the database host, database user, database password and database name to proceed to next step.

    step 6

    Site Config: If the database connection was successful, you need to fill the form with all your details and set some initial system settings.

    step 7

    If every thing goes smooth, You will be redirect to final page with login details.Do not forget to change your password!

  • User Guide

  • User Guide

    Categories

    Categories page will dispaly all the Categories. You can Add, Modify, Delete the Categories from this page.

    Products

    Products page will dispaly all the products and information related to product. You can Add, Modify, Delete the products in this page.

    - Product Code, Product Name, Purchase Price and Price are required. the others is optional if you want or not.

    - Once product added, you can view product, update and delete them anytime.

    - All users can view, update the products but only the admin can delete the products

    - you can also select the product type 'standard', and you will be able to add stock controls and a specific price for every store to your products

    Stores

    you can add as many stores as you want in the store section of the setting page.

    - only the store name is required. the others is optional if you want or not.

    - Once store added, you can view store, update and delete them anytime.

    Warehouses

    you can add your Warehouses in the Warehouse section of the setting page.

    - only the Warehouse name is required. the others is optional if you want or not.

    - Once Warehouse added, you can view Warehouse, update and delete them anytime.

    Customers

    in the Customers page you can add your customers with their contact informations and a specific discount

    - only the Customer name are required.

    Suppliers

    in the Suppliers page you can add your Suppliers with their contact informations

    - only the Supplier name are required.

    POS page

    Sales

    All the sales orders will be dispalyed on this page.

    - You can filter the results using the (barcode in the Receipt) or any other keyword.

    - You can view the sale as an invoice or a Receipt

    - All users can view, update the sales but only the admin can delete them

    Settings

    The setting page will be available only to the admin

    - Company Name : insert you company name (it will be displayed in the invoice and receipt view)

    - Company Logo : upload your company logo

    - Company Phone : add your company phone number

    - Currency Code : Please input your currency code.

    - Default Discount : input your Default order Discount.

    - Defualt TAX : input your Default order TAX.

    - Display virtual keyboard : checked if you want to Display the virtual keyboard in the POS page.

    - timezone : choose your city timezone.

    - Text in the Receipt Header : input text to display in the receipt header (you can drag and drop an image in the textbox).

    - Text in the Receipt Footer : text in the receipt footer (ex: thank you for your business).

    - Pick your theme : choose your prefered theme.

    Users :

    in the user tab you will be able to add, update and delete users informations.

    There are 2 user roles

    - Admins - Admins can perfom all the tasks.

    - Sales Staff - Salesman can add/edit sales/invoices and view total sales, categories and products.

    Repports

    The Repports page will be available only to the admin

    general Reports (customers, product number) (monthly stats in the current yeasr) ... etc

    - Clients or products repprot in a specific Range.

    - Years stats

    Kitchen Page

    this page is for the kitchen user

    the kitchen user will be notified if the table has a new/updated order

    and he can view/print the order

  • how to update to the new versions

    From v1.0.0 to v2.0.0

    ------------------------------------------

    steps

    please don't forget to Backup your files and your DB in case of errors

    - replace the 'application' folder.

    - replace the 'assets' folder.

    - replace the 'sparks' folder.

    - add the 'traits' folder to the main folder.

    - add the 'update' folder to the main folder.

    - add the 'favicon.ico' file to the main folder (or you can add your own favicon).

    - Open the update folder http://yourdomain.com/pathtoupload/update/ in your web browser (in case of error please check the faq page Here).

    - re-enter your database application in the step 2.

    - done !
